Sobre este Projeto

Esse projeto foi desenvolvido para o desafio do programa Jovens Profissionais e 40+ da Minsait.

O desafio proposto foi:

  • Criar uma API REST utilizando o Spring Boot 2.7.4 para permitir uma operação CRUD de clientes e empréstimos;
  • Criar uma Single-Page Application utilizando o Angular 15 e Bootstrap 5.3 para acessar e manipular os clientes fornecidos pela API REST.

Funcionalidades

Aqui serão listadas algumas funcionalidades do sistema:

  • Criação, alteração, leitura e deleção de clientes com CPF, nome, telefone, rendimento mensal, CEP, rua e número;
  • Criação, leitura e deleção de empréstimos com CPF do cliente, valor inicial, valor final, relacionamento, data inicial e data final.
  • Busca automática de endereço via API utilizando o CEP
  • Validação de CPF, CEP e telefone;
  • Bloqueio do campo de CPF ao atualizar;
  • Bloqueio de todos os campos ao deletar;
  • Checagem de existência do CPF ao cadastrar;
